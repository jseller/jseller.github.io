# README #

This is a 3D front end for a blogspot blog. It fetches the atom feed of a blog and displays the labels/categories to navigate by theme. It also shows the title and summary for each blog post.

This uses three.js to create 3D like scenes with the CSS3d library. Works nicely on chrome and firefox, but doesn't draw well on IE.
